Basic script to count words in a whatsapp chat log.
Get the chat logs from Whatsapp in Settings -> Chats and Calls -> Email chat
Change "yourName" to your own name.

Run the script as follows
	run.py [chat file] [name for output]
	eg. run.py chat.txt breakdown

The output will be 2 csv files
	1 with the breakdown of the entire conversation
	1 with the breakdown of only your messages
